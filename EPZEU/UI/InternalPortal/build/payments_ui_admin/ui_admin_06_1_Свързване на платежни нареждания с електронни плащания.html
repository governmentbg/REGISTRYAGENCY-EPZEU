Административен модул

Изход
  • Отбелязване на възстановени надвнесени суми по платежно нареждане
  • Възстановяване на суми от лична сметка
  • Справка по лична сметка
  • Обобщена справка по подадени заявления
  • Справка неплатените такси след направена проверка
  • Банкови сметки на АВ
  • Плащане на задължения по услуги
  • Верифициране на електронни плащания
  • Връщане на суми в лични сметки
